<h1>💰 Projeto: Cálculo de Salários, Vendas e Benefícios</h1>

<p>
  Projeto desenvolvido para praticar <strong>Lógica de Programação</strong> e
  <strong>Programação Orientada a Objetos (POO)</strong> em Java, simulando o cálculo
  de salários, benefícios e desempenho de vendas de funcionários em uma empresa.
</p>

<h2>🎯 Objetivo</h2>

<p>
  Aplicar conceitos fundamentais de <strong>Programação Orientada a Objetos</strong>, como:
</p>

<ul>
  <li>Classes e objetos</li>
  <li>Encapsulamento</li>
  <li>Herança</li>
  <li>Polimorfismo</li>
  <li>Manipulação de listas</li>
  <li>Métodos de cálculo e análise de dados</li>
</ul>

<p>
  O projeto resolve um problema de gestão empresarial relacionado ao cálculo de
  <strong>salários, benefícios e vendas de funcionários</strong>.
</p>

<h2>⚙️ Funcionalidades</h2>

<ul>
  <li>Calcular o <strong>valor total pago aos funcionários</strong> (salários + benefícios) em um determinado mês e ano.</li>
  <li>Calcular o <strong>total pago apenas em salários</strong> no mês.</li>
  <li>Calcular o <strong>total pago em benefícios</strong> para os funcionários que possuem esse direito.</li>
  <li>Identificar <strong>qual funcionário recebeu o maior valor total</strong> no mês.</li>
  <li>Identificar <strong>qual funcionário recebeu o maior valor em benefícios</strong> no mês.</li>
  <li>Identificar <strong>qual vendedor realizou o maior volume de vendas</strong> no mês.</li>
</ul>

<h2>🧩 Estrutura do Projeto</h2>

<p>
  O sistema considera diferentes tipos de funcionários, como:
</p>

<ul>
  <li>Funcionários com salário fixo</li>
  <li>Funcionários com benefícios</li>
  <li>Vendedores com comissão baseada em vendas</li>
</ul>

<p>
  Cada tipo pode possuir <strong>regras específicas de cálculo</strong>, aplicadas
  através de conceitos de <strong>POO</strong>.
</p>

<h2>🛠 Tecnologias Utilizadas</h2>

<ul>
  <li><strong>Java 25</strong></li>
  <li><strong>Programação Orientada a Objetos (POO)</strong></li>
  <li><strong>Coleções (List)</strong></li>
</ul>
